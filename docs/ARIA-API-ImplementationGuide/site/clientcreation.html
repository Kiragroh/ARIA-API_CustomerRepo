VAIS Client Creation

This section describes how to create VAIS client.

Creating a Headless Application

  • This documentation provides a step-by-step guide on how to create a headless application using the provided options. Follow the instructions below to successfully create a headless application:- Open VAIS administration Tool and sign in with your credentials

Basic Details

  • Click on Add Client button to create new client.
  • Enter the desired name for your headless application in the “Basic Details” section.
  • Click on the “Next” button to proceed to the next step.

Fill Secret Details

  • In the “Secret Details” section, select the “SharedSecret” type.
  • Click on the “Add” button to add the shared secret.
  • Once added, click on the “Next” button to proceed to the next step.

Upload API Key File

  • In the “API Keys” section, upload the API key file related to your headless application.
  • After uploading the API key file, click on the “Next” button to proceed further.

Review Details

  • Carefully review all the entered information and settings for your headless application.
  • If everything looks correct, click on the “Submit” button to submit your application.

Upload Additional API Key File

  • Upon successful submission, you will be prompted to upload another API key file.
  • Upload the additional API key file as required and click on the “Next” button.
  • Finally, click on the “Submit” button to complete the process.

Verify Client Creation

  • After submitting the application and completing all the necessary steps, you can now check whether the client has been created successfully with the given name.
