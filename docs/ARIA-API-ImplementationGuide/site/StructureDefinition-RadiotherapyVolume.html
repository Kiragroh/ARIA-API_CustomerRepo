Resource Profile: Radiotherapy Volume

Official URL: http://varian.com/fhir/v1/StructureDefinition/RadiotherapyVolume Version: 1.0.0
Active as of 2024-07-26 Computable Name: RadiotherapyVolume

Copyright/Legal: Varian Medical Systems, Inc.

A volume of the body used in radiotherapy planning or treatment delivery. In ARIA, dose is specified and tracked against Reference Points. These Reference Points can represent actual points or three-dimensional volumes. ARIA calls all of those ‘Reference Point’, even if not a point. Each Reference Point in ARIA is exposed as a Radiotherapy Volume.

Introduction

This profile complies to the CodeX™ RT STU 1 Implementation Guide and is used to implement the integration profiles defined by the (IHE-RO) Exchange of Radiotherapy Summaries (XRTS) Supplement.

The mapping of ARIA to CodeX RT resources is described on the Radiotherapy page.

For the Radiotherapy Volume, see in particular the mapping of Radiotherapy Treatment Sites.

Usage:

  • Refer to this Resource Profile: Radiotherapy Course Summary, Radiotherapy Phase Prescription, Radiotherapy Planned Course, Radiotherapy Planned Phase... Show 3 more, Radiotherapy Treated Phase, Radiotherapy Treated Plan and Radiotherapy Treatment Plan
  • Examples for this Resource Profile: BodyStructure/BodyStructure-PV-681
