Resource Profile: ActivityDefinition

Official URL: http://varian.com/fhir/v1/StructureDefinition/ActivityDefinition Version: 1.0.0
Active as of 2024-07-26 Computable Name: ActivityDefinition

Copyright/Legal: Varian Medical Systems, Inc.

This resource allows for the definition of some activity to be performed, it may be used to specify actions to be taken as part of a workflow, appointment or tasks.

Usage:

  • Examples for this Resource Profile: new Actiity
