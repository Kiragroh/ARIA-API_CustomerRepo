Resource Profile: Appointment

Official URL: http://varian.com/fhir/v1/StructureDefinition/Appointment Version: 1.0.0
Active as of 2024-07-26 Computable Name: Appointment

Copyright/Legal: Varian Medical Systems, Inc.

A booking of a healthcare event among patient(s), practitioner(s), related person(s) and/or device(s) for a specific date/time. This may result in one or more Encounter(s).

Usage:

  • Examples for this Resource Profile: Appointment/Appointment-162
