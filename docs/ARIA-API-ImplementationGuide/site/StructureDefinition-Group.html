Resource Profile: Group

Official URL: http://varian.com/fhir/v1/StructureDefinition/Group Version: 1.0.0
Active as of 2024-07-26 Computable Name: Group

Copyright/Legal: Varian Medical Systems, Inc.

Group represents logical grouping of Resources like Advance Practitioners, Nurses,Oncologists, System administrators etc…
Group comprises Doctor/Staff (Practitioner), Machine (Device) and Venue (Location) with in a department in ARIA.

Usage:

  • Refer to this Resource Profile: ActivityDefinition
  • Examples for this Resource Profile: CareTeamGroup
